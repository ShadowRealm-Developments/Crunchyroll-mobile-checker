## 🚀 Overview
Shadow Realm Crunchyroll Ultimate Checker is a mobile-optimized multi-threaded checker for Crunchyroll streaming accounts. It tests email:password combinations and captures subscription tier (Premium, Expired, Free), saving each category to separate result files with proxy rotation support.

---

## 🎮 Features

### 📺 Capture Features
- Premium subscription detection
- Expired account detection
- Free plan detection
- Subscription plan details
- Proxy rotation support

### ⚙️ System Features
- ⚡ High-speed concurrent engine
- 📡 Proxy support and rotation
- 📊 Live stats per category
- 📁 Separate files per tier (Premium/Expired/Free)
- 🔄 Auto retry on connection errors
